# Code of Conduct

## Our standard

GlossBot is committed to providing a welcoming, respectful, and harassment-free community for everyone.

Examples of behavior that contribute to a positive environment include:

- being respectful in disagreement
- giving constructive, specific feedback
- assuming good intent while still being clear
- helping newcomers understand project context and conventions

Examples of unacceptable behavior include:

- harassment, intimidation, or discrimination
- personal attacks, trolling, or insulting language
- publishing private information without permission
- repeated disruptive or bad-faith behavior

## Scope

This Code of Conduct applies in:

- GitHub issues
- pull requests
- discussions
- project chat or collaboration spaces tied to this repository

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

To report conduct issues, contact the maintainer privately through GitHub rather than escalating in public threads when possible.
